# ITPM_PROJECT (MERN + Vite)

## Setup

### 1) Server

- Create `server/.env` and set:
  - `MONGODB_URI`
  - `JWT_SECRET`
  - `CLIENT_ORIGIN` (default `http://localhost:5173`)

Commands:

- `cd server`
- `npm install`
- `npm run dev`

### 2) Client

Commands:

- `cd client`
- `npm install`
- `npm run dev`

Then open: `http://localhost:5173`
